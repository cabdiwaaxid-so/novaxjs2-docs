Novaxjs2 Framework v9.3.4

Novaxjs2 is a modern, lightweight, and high-performance Node.js framework designed for building web applications and APIs with minimal overhead and maximum flexibility. Version 9.3.4 introduces enhanced performance, improved security features, and extended plugin capabilities.

Whether you're creating RESTful APIs, full-stack apps, or real-time services, Novaxjs2 offers the tools and structure needed without unnecessary bloat. It supports plugins, built-in security features, advanced templating, and custom middleware to help you build fast and secure web experiences.

What's New in v9.3.4

  • Enhanced minification for HTML, CSS, and JavaScript
  • Improved error handling with better stack traces
  • Extended plugin system with more integration points
  • Better performance for template rendering
  • Enhanced security features for file uploads
  • Improved CORS configuration options
  • Updated templating engine with new syntax
